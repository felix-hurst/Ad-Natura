# Extras

The folders in this directory are as follows:

**TechnicalReportSMA:** A Technical Report on the Slime Mold Algorithm (SMA), describing what the SMA is, how it was integrated and implemented into our project, and what behaviors of real life slime molds are simulated in the software.

**UsabilityReport:** The report on the results of the Usability Survey, which seeks to answer whether our project success criteria are met or not.
